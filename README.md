# 栖居设计 · 室内设计工作台

类似酷家乐的网页版室内设计软件：2D 户型编辑 + 3D 实时渲染 + 第一人称漫游。

## 启动

```bash
pnpm install
pnpm dev        # http://localhost:5173
pnpm typecheck  # TS 类型检查
pnpm lint:lines # 工程约束：单文件 ≤ 150 行
pnpm build      # 生产构建
```

## 功能

- **2D 户型编辑**：画墙（吸附/正交/角度）、矩形房间、自动识别房间、门/窗放置、拖拽与端点编辑、尺寸标注
- **玻璃材质**：玻璃墙（铝轨 + 透明玻璃 + 分格条）与玻璃门，2D/3D 双端呈现
- **素材库**：客厅 / 卧室 / 餐厨 / 卫浴 / 桌椅板凳 / 办公家具，办公家具 100+ 款（工位、会议、接待、储物、设备、隔断、茶水间、休闲区…），全部程序化 3D 建模
- **3D 实时渲染**：墙体开洞、程序化地板纹理（木/砖/大理石/地毯）、阴影与环境光、吊顶开关
- **漫游模式**：W/A/S/D 第一人称行走，**靠近门自动播放开门动画**
- **精确输入**：所有尺寸滑杆配数字输入框；墙长 / 家具坐标 / 旋转角可直接键入
- **右键菜单**：旋转 90° / 复制 / 删除 / 玻璃墙(门)切换，2D 与 3D 均可用
- **深浅主题**：暗色 / 亮色一键切换，画布配色联动，跟随系统偏好
- **导入导出**：方案 JSON 导出 / 导入，2D/3D 截图 PNG
- **实时存储**：编辑防抖写入 IndexedDB，刷新自动恢复；撤销 / 重做 100 步
- **示例方案**：温馨两居 74㎡ / 开放一居 LOFT 40㎡（玻璃隔断卧室）/ 现代办公室 112㎡（玻璃会议室）

## 快捷键

`V/W/T/D/N` 工具 · `R` 旋转 · `Del` 删除 · `⌘Z/⇧⌘Z` 撤销重做 · `F` 适配 · `M` 切换 2D/3D · `G` 漫游 · `Esc` 取消

## 技术栈与结构

React 19 + Vite 8 + TypeScript 6 + three.js（pnpm 管理），**全部源码文件 ≤ 150 行**：

```
src/
  core/        类型 / 几何(房间识别) / 素材目录 / 示例 / store(撤销·IndexedDB·导入导出)
  editor2d/    2D 编辑器：输入 / 吸附 / 命中 / 渲染管线 / 家具图例
  viewer3d/    3D 查看器：墙体开洞 / 门窗 / 地板纹理 / 程序化家具 / 漫游开门
  ui/          React 组件：工具栏 / 素材库 / 属性面板 / 右键菜单 / 状态栏
  hooks/       快捷键等
  styles/      主题变量(深/浅) + 布局 + 控件
```
