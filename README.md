# 栖居设计 · Interior Studio

一个运行在浏览器中的专业室内设计工具（类酷家乐）：2D 户型编辑 + 3D 实时渲染，纯前端实现。

线上地址：https://blogs.idevlab.dev/room-design/

## 运行

```bash
npm install
npm run dev        # 开发服务器（默认 http://localhost:5173）
npm run typecheck  # TypeScript 类型检查
npm run build      # 生产构建
```

## 部署

推送到 `main` 分支后，GitHub Actions 会自动构建并发布到 GitHub Pages。

## 功能

- **2D 户型编辑**：连续画墙（端点/墙线/15° 角度/轴对齐吸附）、矩形房间、墙体拖拽与端点编辑、尺寸标注
- **房间自动识别**：墙体围合后自动检测房间多边形（支持 T 形交点、十字交叉拆分），显示名称与面积
- **门窗系统**：墙上单击放置门/窗，可沿墙滑动，宽度/高度/离地高度可调
- **家具库**：客厅/卧室/餐厨/卫浴 27 种参数化家具，2D 俯视图例 + 3D 程序化模型，可旋转/缩放/换色
- **3D 实时视图**：墙体挤出（门洞/窗洞真实开洞）、程序化地板材质（木地板/瓷砖/大理石/地毯）、
  PBR 灯光阴影、家具点选拖拽、第一人称漫游
- **房间属性**：命名、地板材质更换；全局层高、墙厚、墙面颜色
- **工程能力**：撤销/重做、localStorage 自动保存、PNG 截图导出、示例两居室方案

## 快捷键

| 按键 | 功能 | 按键 | 功能 |
| --- | --- | --- | --- |
| V | 选择/移动 | W | 画墙 |
| B | 矩形房间 | D / N | 门 / 窗 |
| R | 旋转家具 90° | 方向键 | 微调位置（Shift ×10） |
| ⌘Z / ⇧⌘Z | 撤销 / 重做 | Del | 删除所选 |
| Enter / Esc | 结束画墙 / 取消 | 空格+拖拽 | 平移画布 |
| 2 / 3 | 切换 2D / 3D | 滚轮 | 缩放 |

## 技术栈

- Vite + TypeScript（无框架，原生 DOM）
- Canvas 2D：户型编辑器
- Three.js：3D 渲染（ExtrudeGeometry 墙体开洞、CanvasTexture 程序化材质、RoomEnvironment 环境光）
- 单位：厘米（cm）；平面坐标 y 轴朝北，3D 映射 `(x, y) → (x, 0, -y)`

## 目录结构

```
src/
  core/        数据模型、几何算法（房间识别）、状态存储、示例工程
  editor2d/    2D 编辑器与家具图例
  viewer3d/    3D 场景、程序化纹理与家具模型
  ui/          工具栏、素材库面板、属性面板
```
